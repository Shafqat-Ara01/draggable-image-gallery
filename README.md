# Draggable Image Gallery

A responsive and interactive image gallery built with HTML, CSS, and JavaScript. This gallery allows users to:

- Filter images by category (Nature, City, Animal, All)
- Drag and drop images to rearrange their positions
- Maintain drag-and-drop functionality even after filtering

## Features

1. **Responsive design** – adjusts layout for mobile, tablet, and desktop screens.
2. **Filter buttons** – dynamically filter images by category.
3. **Drag and drop** – rearrange images freely while maintaining filtered state.
4. **Smooth slide-in animation** – images animate when rendered for a polished UI.
